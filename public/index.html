SCSS

Smart Campus Safety System

Bridging the gap between automated physical security and human reporting. A centralized platform integrating real-time IoT monitoring with institutional incident management.

Core Capabilities

Comprehensive Security Management

Real-time IoT Monitoring

Live tracking of fire exit doors via distance sensors and environmental tracking (DHT11 Temperature & Humidity) powered by ESP32 microcontrollers.

Incident Reporting

Empowering users to submit detailed safety reports directly to the dashboard, including precise locations, severity levels, and photographic evidence.

Direct Messaging

A secure, built-in chat interface facilitating immediate, two-way communication between campus users and the centralized security team during critical events.

Role-Based Access (RBAC)

Strict security separation ensuring that standard users (students/staff) and the security team have optimized interfaces tailored to their specific responsibilities.

About the System

Developed as a Final Year Project for the BSc (Hons) Applied Computing program at the University of Huddersfield. SCSS utilizes a robust tech stack including Node.js, MySQL, and C++ for ESP32 firmware, resulting in a highly reliable and secure environment for modern campus safety.
